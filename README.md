rasterAutocorr
====

Quickly calculate spatial autocorrelation on 2D rasters

## Description
Calculates spatial autocorrelation on a two-dimensional raster* object using fft() methods. Install with: 
```
library(devtools) 
install_github("adammwilson/rasterAutocorr")
```
